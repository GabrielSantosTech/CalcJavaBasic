# Simple Java Calculator

A lightweight Command Line Interface (CLI) application that performs basic arithmetic operations. This project was created to practice Java fundamentals like input handling and control flow.

##  Features

- **Two Input Modes**: Choose between using math symbols (sinais) or words (ptbr).
- **Basic Arithmetic**: Supports addition, subtraction, multiplication, and division.
- **Dynamic Formatting**: Division results are formatted to two decimal places for accuracy.
- **User-Friendly Prompts**: Clear instructions for the user at every step.

##  Technologies Used

- **Language**: Java 8 or higher
- **Core Concepts**: 
  - `Scanner` class for console input.
  - `Switch-case` blocks for operation handling.
  - Type casting (int to double) for precise division.

##  How to Run

1. **Prerequisites**: Ensure you have [JDK](https://www.oracle.com/java/technologies/downloads/) installed on your machine.
2. **Download**: Copy the `baseMain.java` file to your local machine.
3. **Compile**: Open your terminal or command prompt and run:
   ```bash
   javac baseMain.java
